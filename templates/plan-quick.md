# Plan: Quick

For requests that can be scoped and executed in one session. No formal research step needed.

## When to Use

- A single-surface post or short email
- A bug fix or small feature with clear requirements
- A question with a known answer

## Template

```
QUICK PLAN — [title]
Date: [date]

What:
[One sentence describing the deliverable]

Audience:
[Who it's for]

Key message / outcome:
[The one thing that must land]

Constraints:
[Length, tone, platform, technical limits]

Assigned to:
[Specialist or direct execution]

Done when:
[Clear success criterion]
```

## Process

Orchestrator fills the template, routes to specialist (or executes directly), confirms when done. No pipeline steps.
