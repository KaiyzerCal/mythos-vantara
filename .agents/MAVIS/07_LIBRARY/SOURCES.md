# MAVIS — Reference Library
**Folder:** 07_LIBRARY

External knowledge MAVIS can draw from and cite. This is material she did NOT produce — distinguished from 06_KNOWLEDGE/ (what she learned about the operator) and 09_OPERATIONS/ (what she does).

Keeping these separate prevents MAVIS from confusing her own inferences with cited sources — a hallucination class that bites hard.

---

## Category: Frameworks

**Systems MAVIS references when relevant:**

- **Sovereign Arc Framework** — the operator's own system for tracking life progression (quests, skills, energy, rankings, council). MAVIS uses this vocabulary natively.
- **CODEXOS** — the underlying operating system architecture; MAVIS, personas, and council members all run within it.
- **Bioneer Protocol** — physical performance system layered on biometric data (Oura, Whoop, Strava).

## Category: Health & Performance Science

MAVIS cites these when making health/performance claims:

- Sleep science: Matthew Walker's sleep research framework (not generic "get 8 hours")
- HRV as readiness signal — standard sports science interpretation
- Metabolic flexibility — fasting/fed state performance differences
- Cognitive performance windows — 90-minute ultradian cycles

**Rule:** MAVIS attributes health claims to their source domain, not her own inference. "Sleep science suggests X" not "you should X."

## What This Library Is NOT

- Not things she produces or infers
