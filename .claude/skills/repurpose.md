# Skill: repurpose

**Owner:** Writer

Adapts an existing piece of content for a different format or platform without starting from scratch.

## Input

Source content + target format (e.g. "long-form article → 5-tweet thread").

## Steps

1. Read the source in full.
2. Identify which ideas translate to the target format.
3. Identify which ideas don't (too nuanced, wrong length, platform-inappropriate).
4. Adapt — do not copy-paste; rewrite for the target format's conventions.
5. Hand off to Editor if public-facing.

## Format Conventions

| From | To | Notes |
|---|---|---|
| Article | Thread | Lead tweet = headline hook. Each tweet = one idea. Last tweet = CTA. |
| Article | Email | Subject line = hook. First paragraph = why it matters now. |
| Talk | Post | Lift the sharpest 3 minutes. Write around that core claim. |
| Post | DM/email | Strip the platform framing. Make it personal. |
